# Introduction 
This a bash script to can analyse basic server performance stats. 

1. Uptime
2. Total CPU usage
3. Total memory usage 
4. Total disk usage 
5. Top 5 processes by CPU usage
6. Top 5 processes by memory usage

## Usage
- chmod +x server-stats.sh
- ./server-stats.sh
